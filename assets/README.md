# Assets for Delay Discounting Task

This protocol does not require external image/audio/video files.

All trial stimuli (amount and delay texts) are rendered at runtime via PsychoPy text stimuli
and mapped to literature evidence in `references/stimulus_mapping.md`.

If a future protocol revision requires external assets, update:
